# 📝 Flask To-Do App  

A Microsoft To-Do clone built with **Python Flask**, featuring user authentication, task management, and a clean UI.  
 

## ✨ Features  
- ✅ **User Authentication** (Login/Register)  
- 📋 **Multiple Task Lists**  
- ⏰ **Due Dates & Reminders**  
- ⭐ **Priority Marking** (Important Tasks)  
- 📱 **Responsive Design**  
- 🔒 **CSRF Protection**
